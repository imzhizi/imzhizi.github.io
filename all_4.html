225 Implement Stack using Queues

题目描述

Implement the following operations of a stack using queues.

push(x) -- Push element x onto stack.
pop() -- Removes the element on top of the stack.
top() -- Get the top element.
empty() -- Return whether the stack is empty.

代码实现

详解见注释

/**
 
<!-- more -->
 
 * created by zhizi on 11/13/18
 */
class MyStack {
    /**
     * 题目分析：
     * 栈是一种非常基本的数据结构，特点就是先进后出之类的
     * 用一个数组加一个指针可以轻松模拟栈的基本操作
     *
     * 运行时长：
     *
     * 总结：
     * 基础
     */
    int[] stack;
    int top=-1;

    /** Initialize your data structure here. */
    public MyStack() {
        stack = new int[100];
    }

    /** Push element x onto stack. */
    public void push(int x) {
        stack[++top] = x;
    }

    /** Removes the element on top of the stack and returns that element. */
    public int pop() {
        return stack[top--];
    }

    /** Get the top element. */
    public int top() {
        return stack[top];
    }

    /** Returns whether the stack is empty. */
    public boolean empty() {
        if (top==-1) return true;
        else return false;
    }


    public static void main(String[] args) {
        MyStack stack = new MyStack();

        stack.push(1);
        stack.push(2);
        System.out.println(stack.top());   // returns 2
        System.out.println(stack.pop());   // returns 2
        System.out.println(stack.empty()); // returns false
    }
}

Win10 MySQL zip安装方法

1、下载，到MySQL官网：https://dev.mysql.com/downloads/mysql/

2、解压安装包
解压下载的安装包，放到你的电脑路径，我放置的是 E:\mysql

3、配置my.ini文件
解压后，在根目录下创建配置文件my.ini，内容如下：

[mysql]
; 设置MySQL客户端默认字符集
default-character-set=utf8
[mysqld]
;设置3306端口
 
<!-- more -->
 
port = 3306
; 设置MySQL的安装目录，应该与解压位置相匹配
basedir=E:\mysql
; 设置MySQL数据库的数据的存放目录
datadir=E:\mysql\data
; 允许最大连接数
max_connections=200
; 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
; 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB

4、配置MySQL环境变量
为了以后方便启动mysql，在环境变量Path追加：E:\mysql\bin

5、安装MySQL
以管理员身份打开cmd窗口（开始菜单下输入cmd，右键管理员方式运行）

  • 输入：mysqld --initialize-insecure，安装需要等待一会
    这一步用于初始化MySQL的data文件夹，官方的压缩包解压后并没有data文件夹，运行命令后，按照my.ini中配置的位置生成data文件；
  • 输入：mysqld install 会提示安装成功；
  • 输入：net start mysql 启动服务，若显示启动成功则安装成功；
  • 最后输入：net stop mysql，停止服务，以便进行以下操作。

6、设置root管理员密码

  • 在cmd中输入 mysql -u root -p

  • 此时cmd显示 Enter password： 不必理会，直接回车即可成功进入MySQL，进入后使用以下方法修改密码

    • SET PASSWORD for root@localhost = PASSWORD("root"); (5.7版本修改密码）
    • ALTER USER root@localhost IDENTIFIED BY "root"; (8.0版本修改密码）

  • 以后就可以通过在cmd中输入 mysql -u root -p 进入MySQL

至此，MySQL就安装完成并可以使用了。

忘记密码

  • 以系统管理员身份运行cmd
  • 查看mysql是否已经启动，如果已经启动，就使用 net stop mysql 停止服务
  • 在命令行输入 mysqld -nt --skip-grant-tables
  • 此时通过 mysql -u root -p 就可以直接进入MySQL
  • 进入后需要设置密码，因为mysql会仍以为你是第一次进入MySQL
  • 输入之前设定重新确定密码

PS：大部分SQL语句需要加分号，可以使用quit或者exit来退出MySQL
但是，在日常使用中，建议使用权限更低的用户来操作具体的数据库，SQL语句如下：

create database xxx;
CREATE USER zhizi IDENTIFIED BY '111';
grant all privileges on xxx.* to zhizi@'localhost' identified by '111';
