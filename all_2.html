Week - 2018/12/15

Memo

基金

前些日子买了一些基金, 很少的一点, 纯粹是玩一玩, 了解一下基金是什么, 具体是怎么赚到钱的. 虽然说基金是通过长期持有来获得收益的, 但到目前为止, 我买的基金大部分都处于亏损状态 😢.

在这过程中我确实翻了一些很基础的错误, 其中的一个就是由于基金和股票的相关性, 所以对于国内的基金交易, 下午三点就会闭市, 闭市之前的交易都算做当天的, 也就是会按照当天的价格计算, 而过了三点, 价格就会按照第二天三点的价格计算, 所以一般的交易都发生在三点前的几分钟, 这时候可以根据股票的走势来决定买入或者卖出.

另外一方面, 我总是对市场抱有一些幻想, 或者说挺贪的, 不能果断地做出决断, 所以一些还不错的卖出机会我都没有抓住, 才迎来了最近持续了很久时间的亏损.

macOS 安装多个 Java 版本

安装说明

Homebrew 是 macOS 下的一个非常好用的包管理工具, caskroom 则是基于 Homebrew 构建的一个强大的应用程序管理器.

$ brew tap homebrew/cask-versions
$ brew cask install java8
# 需要输入 root 密码以完成安装
# 输入 java -version 测试安装是否成功
$ java -version
java version "1.8.0_192"
Java(TM) SE Runtime Environment (build 1.8.0_192-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.192-b12, mixed mode)

# 需要先把 caskroom/versions 加入下载源
