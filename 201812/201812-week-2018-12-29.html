Week - 2018/12/29

李铭一
2018-12-28
memo, 周记

Me

可能北方比较干燥的原因, 入秋以来经常被静电电到, 尤其是金属外壳、两脚插头、没有地线的 Mac, 睡了午觉回到实验室, 掀开电脑的瞬间, 心跳加速
手指抽搐, 一发入魂. 后来每次我掀开电脑时总是像摸老虎一样特别紧张, 但紧张也没用, 该电还是会被电. 我调查了许多方法, 比如常常洗手什么的, 感觉效果都不好. 其实核心问题就是在用电脑之前把身上的静电释放掉, 所以找一个良好的导体很重要, 在我寻寻觅觅, 找了几天之后, 终于终到了一个非常合适的导体 —— 厕所的下水管, 那是个敦实、粗大的金属水管. 每次我只要把手掌整个地覆盖在水管上, 冰冰凉凉的触感让我无比的安心, 效果确实不错, 自此我很少被电到.

基金依旧在亏, 其中很让我觉得有意思的就是日常看到的那些新闻, 都会成为影响基金涨跌的催化剂? 比如游戏开始发放版号, 游戏相关股票基金直接就见涨. 比如之前国家公布了医药相关的政策, 要降低药价之类的, 医药相关就闻声而降, 以前从来没感觉事物之间的联系是如此紧密, 看来利益让人做到了.

Movie

给室友推荐了几次电影之后, 他似乎很满意, 他是个执行力很强的人, 每次我推荐完之后他都立刻到 bt 上去下载, 三天内必看掉, 这周我推荐了《天注定》、《狩猎》.

而我这周只尝试了下新出的电视剧《古董局中局》, 剧情进展太慢了, 看得不太有意思. 我一直在尝试把豆瓣上标记的想看给看掉, 所以这周看了《新宿事件》和《极速风流》,《极速风流》其实还不错, 名字翻译得太烂了. 我的努力很明显, 这次豆瓣的年度总结是有这样一个数据, 感觉自己还不错.

Words

Derek Sivers: Keep your goals to yourself | TED Talk

大意是说, 如果想要目标实现, 最好不要告诉别人, 蛮好玩的.

New

这周重新尝试了「番茄土豆」App, 主要原因还是之前自己折腾了很多方法论、但很多时候只是了解, 并不是真的为了解决自己的问题. 这次想重新开始. 一方面受朋友的影响, 一方面看到了这篇文章 经过 1700 个番茄时间，番茄工作法改变了我什么 - 少数派, 所以再一次对生产力这件事情产生幻想 🙃.

Skill

  • Git 里对于已经 commit 的代码, 如果想反悔, 可以用这个命令撤销commit git reset HEAD .
    Git 笔记 - 难以想象的晴朗

  • 还是一个跟 Git 相关的, 了解了 Unix 里 diff 命令输出结果应该怎么理解.

变动的位置用两个 @ 作为起首和结束。@@ -1,8 +1,8 @@
前面的分成三个部分：减号表示第一个文件(即 tt), “-1,8”表示下面是第一个文件从第 1 行开始的连续 8 行. 同样地，”+1,8”表示变动后，成为第二个文件从第 1 行开始的连续 8 行。理解 Mac-diff 指令 - 陈斌彬的技术博客 )

  • 在使用 Windows Server 时, 如果整个项目都已经规定了 UTF-8, 但数据保存到数据库还是成了乱码, 可以考虑设置一下在 datasource url 增加 useUnicode=true&characterEncoding=UTF-8
