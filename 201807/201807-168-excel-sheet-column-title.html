#LeetCode 168. Excel Sheet Column Title

质子
2018-07-13
algs, leetcode

Excel表格的列标题

题目描述

Given a positive integer, return its corresponding column title as appear in an Excel sheet.
For example:
1 -> A
2 -> B
…
26 -> Z
27 -> AA

Example 1:
Input: 1
Output: “A”

Example 2:
Input: 701
Output: “ZY”
